# Contributing

Thanks for your interest in improving this repository.

## Scope

This repository is a practical framework for AI release governance. Helpful contributions improve clarity, rigor, and operational usefulness.

## Good contributions

Examples of helpful contributions include:
- improving framework clarity,
- refining risk and gate definitions,
- adding realistic examples,
- improving templates or supporting documentation,
- correcting inaccuracies.

## Contribution expectations

Please keep contributions:
- concise,
- implementation-aware,
- evidence-oriented,
- free of proprietary or confidential material.

## Pull requests

When opening a pull request, explain:
- what changed,
- why it improves the repository,
- and whether it affects the framework, examples, code, or templates.
